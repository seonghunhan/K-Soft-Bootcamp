
test.sh파일 권한거부당하면 ->  chmod +x file_name 이거해주면 ls칠때 초록불뜨면서 ./test.sh 하면 안에
내용물 출력함

.sh파일을 만들어서 그안에 커맨드들 리스트를 넣고
chmod +x로 권한을 주고 실행싴면 명령어들을 한번에 처리가 가능함

echo '1234' -> 1234를 화면에서 echo 해준


ls -l -> 현재 위치에 저장된 파일 리스트지만
          각각의 파일들의 권한, 소유자, 생성 날짜, 파일 크기들 볼수 있게 해줌

chmod -> 파일 권한 바꿔줌

cat -> 그 문서의 내용을 화면에서 보여줌

mkdir 폴더네임 -> 폴더 만들어줌(색깔다름)

mv 1232 ./testdir/1232 -> 파일을 옮기는데 사용
mv 파일이름 파일이름 -> (같은위치라면) 파일 이름바꾸는데도 사용함

cp 1234 1235 -> 1234를 1235라는 이름으로 파일 카피

cd -> 폴더 이동하기

pwd -> 현재 위치
rm 파일이름 -> 삭제

df ->

make -> make파일 구동
make clean -> make파일로 인해 만들어진 파일들 삭제

** OS 종류는 크게 윈도우, 리눅스, 유닉스, 맥OS이다
** 리눅스에는 크게 레드햇, 데비안, 우분투 3가지이다
** 여기서 리눅스 패키지(apt, dpkg, yum, rpm) 비교를 하자면
레드햇계열은 yum사용
데비안, 우분투 계열은 apt-get을 사용한다.
